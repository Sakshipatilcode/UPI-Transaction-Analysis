# UPI Transaction Analysis 

##  Project Overview
Analyzed 2,50,000 UPI transactions to find
patterns in digital payments, fraud detection
and user behavior across India.

##  Tools Used
| Tool | Purpose |
|------|---------|
| Excel | Data Cleaning + Dashboard |
| Power BI | Interactive Dashboard |
| MySQL | Data Querying |
| Python | Data Visualization |

##  Key Business Insights
-  Total Amount: ₹32,79,39,009
-  Success Rate: 95.05%
-  Fraud Transactions: 480 (0.19%)
-  Top State: Maharashtra
-  Most Used Device: Android
-  Peak Hour: 19hr
-  Avg Transaction: ₹1,311.76

##  Files in Repository
- Cleaned Excel Data
- Power BI Dashboard (.pbix)
- Python Notebook (.ipynb)
- SQL Queries (.sql)
- Dashboard Screenshots

##  How to Run
1. Open .xlsx in Microsoft Excel
2. Open .pbix in Power BI Desktop
3. Run .sql in MySQL Workbench
4. Open .ipynb in VS Code

##  Why This Project
UPI has revolutionized digital payments
in India. This analysis helps understand
payment patterns, fraud detection and
state wise adoption of UPI technology.
